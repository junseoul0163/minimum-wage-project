# Raw Data

This folder contains raw CBP data files. Due to size limits, they are not stored on GitHub.

## How to download:

1. Go to: https://www.census.gov/programs-surveys/cbp/data/datasets.html
2. Click on each year (2015, 2019, 2023)
3. Download "County File" for each year
4. Place the .txt files in this folder
